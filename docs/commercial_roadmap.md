# EduLens AI — Commercial Roadmap

Prototype pricing for product discussions. **Not live billing.**

## EduLens tiers

| Tier | Price (SGD) | Users | Features |
|------|-------------|-------|----------|
| Student | $0 | 1 | 5 homework analyses / month |
| Student Plus | $3.99 / mo | 1 | Unlimited analyses, mistake book |
| Classroom | $12 / student / year | Teacher + class | Lesson generator, dashboard, PDF reports |

## Value proposition

- Save teacher marking time on draft feedback
- Give students structured mistake patterns
- Syllabus-aware lesson generation for MOE-aligned content

## Cost assumptions

| Item | Estimate |
|------|----------|
| Vision + text LLM per analysis | ~$0.02–0.08 |
| RAG indexing | One-time local script |
